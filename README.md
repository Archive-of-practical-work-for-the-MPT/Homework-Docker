# Практическая работа № 9 - Работа с InfluxDB
  
## Цель

1. Необходимо перевести кастомные метрики, созданные на практической работе №7 в формат InfluxDB согласно официальным примерам  InfluxDB и визуализировать на дашборде в Grafana необходимо присоединить данные именно через источник InfluxDB). 
2. Изучить язык написания запросов Flux, так как он вам потребуется в запросах. На самой InfluxDB есть примеры всех функция.
3. Хотя бы одна метрика должна иметь Тег, который реально необходим, а не просто взят из воздуха.
4. Необходимо присоединить плагин Telegraf,и с помощью него вывести данные из любого сервиса, не рассмотренного на паре.
