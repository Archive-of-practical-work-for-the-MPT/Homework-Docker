# Практическая работа № 7 - Отслеживание метрик с собственного приложения
  
## Цель

1. Необходимо создать отслеживание метрик с вашего приложения. Данные также должны визуализироваться с помощью Grafana.
2. Выгрузить данные по работе приложения в формате Prometheus
3. Визуализировать 3 любые метрики, которые важны для анализа работоспособности приложения
4. Визуализация должна быть логичной и подходить данной метрике
5. Легенда должна быть настроена на вывод основных данных
6. Создать 3 собственные кастомные метрики по работе конкретно вашего приложения ( например, количество пользователей на сайте или количество данных в БД)
7. Визуализировать собственные метрики на дашборде
8. То, что вы отслеживаете - должно являться частью вашей [курсовой](https://github.com/Archive-of-practical-work-for-the-MPT/GreenQuality) (например API или Веб приложение, Десктоп приложение (да, даже из него можно выгружать метрики))
