## Цель

Необходимо переделать pipeline GitLab для GitHub Actions с сохранением всех этапов, зависимостей. Версии Actions нужно использовать самые последние. В качестве Runner’а использовать Ubuntu.

## Как сделать чтобы работало?
- Склонируйте репозиторий
- Запушьте в свой репозиторий, pipeline подтянется автоматически
