# Практическая работа № 6 - Настройка отслеживания базы данных с помощью, локально развернутой в Docker Grafana

## Цель

1. Описание развертывания локальной графаны на своем устройстве в Docker.
2. Принцип добавления новых источников получения данных
3. Развернуть БД в отдельном docker-compose файле. Настроить единую с Grafana сеть.
4. Создание выгрузки данных на дашборд из базы данных. Необходимо выгрузить данные и с хост машины (именно на прямую, а не через Prometheus), и из развернутой в докере в единой сети с графаной БДшки.
5. Настроить выгрузку данных любой другой БД кроме Postgresql
6. Как минимум 1 столбик должен иметь вид даты и 1 столбик иметь числовые данные. Визуализировать эти данные в виде графика.
